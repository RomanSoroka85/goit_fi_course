Модуль 1, задание 1

Создать 2 маркированных списка: первый должен содержать li, внутри которых будут имена строчных тегов изученных в модуле. Второй должен содержать li, внутри которых будут имена блочных тегов изученных в модуле.

Строчные элементы:

  • "a" - предназначен для создания ссылок.
  • "img" - для вставки картинки.
  • "span" - для вставки текста.
  • "br" - перенос строки.
  • "textare" - для вставки большого контейнера
  • "script" - позволяет подключить java script на наш сайт
  • "b" - устанавливает жирное начертание шрифта.
  • "big" Увеличивает размер шрифта на единицу по сравнению с обычным текстом.
  • "i" - устанавливает курсивное начертание шрифта
  • "small" - уменьшает размер шрифта на единицу по сравнению с обычным текстом.
  • "strike" - отображает текст как перечеркнутый.
  • "strong" - акцентирует текст, обычно жирным начертанием.
  • "u" - добавляет подчеркивание к тексту.

Блочные элементы:

  • "div" - Универсальный блочный элемент.
  • "form,input tupe,inpit submit" - форма с возможностью заполнения либо отправки.
  • "h1,h2,h3,h4,h5,h6" - заголовок с 1го по 6й уровень с размером от большего к меньшему.
  • "hr" - рисует горизонтальную линию.
  • "оl,li" - контейнер списка с нумерованными пунктами.
  • "p"- абзац текста.
  • "ul,li" - контейнер списка с ненумерованными пунктами.
  • "table,tr,td" - таклица с указанием строк и ячеек.
  • "menu" - отображает список меню.
  • "fieldset" - предназначен для группирования элементов формы.
  • "center" - выравнивает содержимое контейнера по центру относительно родительского элемента.
